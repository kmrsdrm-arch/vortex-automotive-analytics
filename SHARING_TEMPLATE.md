**Q: Why Streamlit instead of React?**
A: Streamlit allowed rapid prototyping and is perfect for data-heavy dashboards.

**Q: How do you handle OpenAI API costs?**
A: Implemented caching for repeated queries, use appropriate models (GPT-3.5 for

**Q: What about security?**
A: API keys stored as environment variables, Pydantic validation on all inputs, 
SQLAlchemy ORM prevents SQL injection, CORS properly configured.

**Q: How would you scale this?**
A: Add Redis caching, implement load balancing, use CDN for static assets, 
optimize database queries with indexes, move to managed Kubernetes for orchestration.
